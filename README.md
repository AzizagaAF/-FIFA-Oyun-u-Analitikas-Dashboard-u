# -FIFA-Oyun-u-Analitikas-Dashboard-u
Bu layihə, dünya üzrə futbolçu datalarından istifadə edərək, oyunçuların fiziki xüsusiyyətləri, bazar qiymətləri və karyera potensialları arasındakı əlaqələri vizuallaşdıran interaktiv bir dashboard-dur.
## 📊 Layihənin Məqsədi
Məlumat analizi (Data Analysis) və vizuallaşdırma üsullarından istifadə edərək aşağıdakı suallara cavab tapmaq:
* Oyunçuların mövqelərinə görə paylanması necədir?
* Beynəlxalq reputasiya oyunçunun potensialına və bazar qiymətinə necə təsir edir?
* Sağ və sol ayaqlı oyunçular arasında qiymət və inkişaf trendi baxımından fərq varmı?
* Yaş paylanması və oyunçu sıxlığı hansı aralıqdadır?

## 🛠 İstifadə Olunan Texnologiyalar
* **Python**
* **Pandas** (Məlumatların manipulyasiyası)
* **Matplotlib** & **Seaborn** (Qabaqcıl vizuallaşdırma)

## 📈 Dashboard-da əks olunan analizlər
1. **Mövqe Analizi:** Hansı mövqedə daha çox oyunçu var.
2. **Yaş Paylanması:** Histoqram və KDE vasitəsilə yaş sıxlığının analizi.
3. **Maliyyə Analizi:** Mövqe və üstünlük verilən ayağa görə orta bazar qiymətləri.
4. **Potensial Müqayisəsi:** Beynəlxalq reputasiyanın potensiala təsiri (Boxplot & Stripplot).
5. **İnkişaf Trendi:** Hazırkı güc (Overall) və Potensial arasındakı korrelyasiya.
